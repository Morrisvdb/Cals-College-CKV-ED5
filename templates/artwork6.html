{% endblock %} {% block header %}The Creeper{% endblock %} {% block description %} The Creeper (also known as 'The Creeper and Reaper') was a Proof of concept computer virus that was written by Bob Thomas in 1971. It was the first computer virus to be written and was created as an experiment to see if it was possible to create a self-replicating program. The Creeper was written in the programming language B5500 and was designed to infect DEC PDP-10 computers running the TENEX operating system. The virus would display the message "I'm the creeper, catch me if you can!" on the infected computer's terminal and then move to another computer on the network. The Creeper was not malicious and was not designed to cause any harm to the infected computers. It was simply a demonstration of the concept of self-replicating programs. The Creeper was later followed by the first computer worm, the Morris worm, which was written by Robert Tappan Morris in 1988. The Morris worm was the first computer virus to spread over the Internet and caused widespread damage to computer systems around the world. The Creeper and the Morris worm are considered to be the first examples of malware, or malicious software, and are the precursors to the computer viruses, worms, and other types of malware that we see today.
